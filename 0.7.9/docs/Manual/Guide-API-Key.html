    Introduction

    The API key used in the Sensorberg SDK is a unique hash string which is automatically created when you add a new app to the Sensorberg Beacon Management Platform.

    This guide describes how to retrieve and integrate the API key into your iOS apps.

    This document contains the following sections:

    • Retrieving the API key
    • Embedding the API key into your code

    Retrieving the API key

    1. Before being able to retrieve the API key, you must create an account at the Sensorberg Beacon Management Platform.

    2. Sign in.

    3. In the Apps section you can retrieve the API key for an existing application or create a new API key.

    Embedding the API key into your code

    1. Open your AppDelegate.m file.

    2. Search for the method application:didFinishLaunchingWithOptions:

    3. Modify the following line of the bootstrap of the Sensorberg SDK:

       [self.beaconManager connectToBeaconManagementPlatformUsingApiKey:@"<API-KEY>" error:&connectionError];

    4. Replace <API-KEY> with the API key associated with your app.
