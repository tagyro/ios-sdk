beaconManager:didDetectBeaconEnterEventForBeacon:

Delegate method invoked when a beacon region has been entered and is about to be resolved.

- (void)beaconManager:(id)manager didDetectBeaconEnterEventForBeacon:(id)beacon

Parameters

manager

Beacon manager.

beacon

Detected beacon.

Discussion

This delegate method is being called for each single detected beacon that is part of the beacons regions.

It will only resolve into a beacon action if the specific beacon has an active campaign at the Sensorberg Beacon Management Platform.
